  - OriginAirportID
  - OriginAirportSeqID
  - OriginCityMarketID
  - Origin
  - OriginCityName
  - OriginStateName
- Destination:
  - DestAirportID
  - DestAirportSeqID
  - DestCityMarketID

  - DepDelay
  - DepDelayMinutes
  - DepDel15

  - ArrDelayMinutes
  - ArrDel15
- Cancellations and Diversions:
  - Cancelled
  - CancellationCode
  - Diverted
- Flight Summaries

- Cause of Delay
